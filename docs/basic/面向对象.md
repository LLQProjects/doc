## 变量空间 - 对象空间

```js
const myObj = {name: 'hello'}; // myObj 称为变量空间，存在栈中。{name: 'hello'}称为对象空间，存在堆中
const newObj = myObj;// 把引用地址复制给newObj
myObj = {name: 'world'}; // myObj指向堆中新建的 {name: 'world'};
newObj.name // hello
```
